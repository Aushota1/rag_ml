## Структура данных для обучения

Классификаторам нужны данные в следующем формате:

### Document Classifier
```
data/
├── laws/          # Законы
│   ├── doc1.pdf
│   └── doc2.pdf
├── cases/         # Судебные дела
│   ├── case1.pdf
│   └── case2.pdf
└── regulations/   # Регуляции
    ├── reg1.pdf
    └── reg2.pdf
```

### Question Classifier
Нужен файл `data/questions.json`:
```json
[
  {
    "question": "What is the penalty?",
    "type": "factual"
  },
  {
    "question": "How to apply?",
    "type": "procedural"
  }
]
```

### Relevance Classifier
Нужны пары документ-вопрос с метками релевантности.
