# Настройка LLM для генерации ответов

## Быстрый старт

Система теперь поддерживает использование мощных LLM моделей вместо простых эвристик.

### 1. Настройка переменных окружения

Создайте файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

### 2. Добавьте API ключ OpenAI

Отредактируйте `.env` и добавьте ваш API ключ:

```env
USE_LLM=true
LLM_PROVIDER=openai
LLM_MODEL=gpt-4o-mini
OPENAI_API_KEY=sk-your-actual-api-key-here
```

### 3. Доступные модели

#### OpenAI модели (рекомендуется):
- `gpt-4o-mini` - быстрая и экономичная (по умолчанию)
- `gpt-4o` - более мощная
- `gpt-4-turbo` - баланс скорости и качества
- `gpt-3.5-turbo` - самая быстрая и дешевая

#### Отключение LLM:
Если хотите использовать эвристики (без API ключа):
```env
USE_LLM=false
```

### 4. Запуск генерации

```bash
python hack/generate_submission.py
```

Система автоматически:
- Загрузит вопросы из `questions.json`
- Использует выбранную LLM модель для генерации ответов
- Сохранит результаты в `hack/submission.json`
- Укажет имя модели в телеметрии

## Преимущества LLM

✅ Более точные ответы на сложные вопросы
✅ Лучшее понимание контекста
✅ Корректная обработка всех типов ответов (boolean, number, date, name, names, free_text)
✅ Автоматическое форматирование ответов

## Стоимость

Примерная стоимость для 100 вопросов с `gpt-4o-mini`:
- ~$0.10-0.30 USD (зависит от длины контекста)

## Troubleshooting

**Ошибка: "openai package not installed"**
```bash
pip install openai
```

**Ошибка: "Invalid API key"**
- Проверьте правильность API ключа в `.env`
- Убедитесь что у вас есть баланс на аккаунте OpenAI

**Fallback на эвристики**
Если LLM недоступна, система автоматически переключится на эвристическую генерацию.
