### 2. Два режима индексации ✅

**Standard Mode:**
```bash
python build_index.py --mode standard
```
- Быстрое построение (2-5 минут)
- Структурная сегментация
- Не требует обучения моделей
- Базовое качество

**Tokenier Mode:**
```bash
python build_index.py --mode tokenier
```
- Классификация типов документов
- Семантическая сегментация
- Улучшенное качество (+5-10%)
- Требует обучения моделей

### 3. Полезные скрипты ✅

**check_tokenier_setup.py** - Проверка настройки
```bash
python check_tokenier_setup.py
```
- Проверяет наличие файлов
- Тестирует токенизатор
- Проверяет эмбеддинги
- Показывает статус обученных моделей

**compare_indexing_modes.py** - Сравнение режимов
```bash
python compare_indexing_modes.py
```
- Строит индексы в обоих режимах
- Сравнивает скорость
- Сравнивает количество чанков
- Сохраняет результаты в JSON

**train_tokenier_models.py** - Обучение моделей
```bash
python train_tokenier_models.py --model all
```
- Обучает все классификаторы
- Поддержка выборочного обучения
- Автоматическая валидация

## 🚀 Как использовать

### Быстрый старт (5 минут)

```bash
# 1. Установка
pip install -r requirements.txt

# 2. Настройка
cp .env.example .env
# Отредактируйте .env

# 3. Построение индекса
python build_index.py --mode standard

# 4. Запуск
python api.py
```

### С tokenier (30 минут)

```bash
# 1. Проверка файлов
python check_tokenier_setup.py

# 2. Обучение моделей
python train_tokenier_models.py --model all

# 3. Построение индекса
python build_index.py --mode tokenier

# 4. Сравнение режимов (опционально)
python compare_indexing_modes.py
```

## 🐛 Troubleshooting

### Проблема: "Tokenizer not found"
**Решение:**
```bash
python check_tokenier_setup.py
# Следуйте инструкциям
```

### Проблема: "No models trained"
**Решение:**
```bash
python train_tokenier_models.py --model all
```

### Проблема: Медленное построение
**Решение:**
```bash
# Используйте standard mode
python build_index.py --mode standard
```

### Проблема: Ошибка импорта
**Решение:**
```bash
pip install -r requirements.txt
```
