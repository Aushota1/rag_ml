## Возможности

### Типы визуализаций

1. **3D Scatter Plot** - интерактивная 3D проекция эмбеддингов
   - UMAP (рекомендуется)
   - PCA (быстрый)
   - t-SNE (для кластеров)

2. **2D Scatter Plot** - плоская проекция для быстрого анализа

3. **Distribution Analysis** - статистическое распределение:
   - Нормы векторов
   - Средние значения
   - Стандартные отклонения
   - Корреляции

4. **Dimension Analysis** - анализ по измерениям:
   - Среднее по измерениям
   - Разброс по измерениям
   - Min/Max значения
   - Variance по измерениям

### Интерактивность

- 🔄 Вращение 3D графика
- 🔍 Зум и перемещение
- 📊 Hover для деталей
- 💾 Экспорт в HTML
- 🎨 Цветовая шкала по норме

## Быстрый старт

### Полный отчет (все визуализации)

```bash
python visualize_embedding_model.py --full-report
```

Создаст 5 HTML файлов в `visualizations/`:
- `3d_umap.html` - 3D UMAP проекция
- `3d_pca.html` - 3D PCA проекция
- `2d_umap.html` - 2D UMAP проекция
- `distribution_analysis.html` - Статистический анализ
- `dimension_analysis.html` - Анализ по измерениям

### Одна визуализация

```bash
# 3D UMAP (по умолчанию)
python visualize_embedding_model.py

# 3D PCA (быстрее)
python visualize_embedding_model.py --method pca

# 2D визуализация
python visualize_embedding_model.py --2d
```

## Параметры

```bash
python visualize_embedding_model.py \
  --model models/tokenier/embedding_model.pth \  # Путь к модели
  --output-dir visualizations \                   # Директория вывода
  --method umap \                                 # umap/pca/tsne
  --sample-size 5000 \                           # Количество точек
  --full-report \                                # Все визуализации
  --2d                                           # 2D вместо 3D
```

## Что показывают визуализации

### 3D/2D Проекции
- Каждая точка = токен из словаря
- Цвет = норма вектора (яркость эмбеддинга)
- Близкие точки = семантически похожие токены
- Кластеры = группы связанных токенов

### Distribution Analysis
- **Norms** - как распределены нормы векторов
- **Means** - среднее значение компонент
- **Stds** - разброс значений
- **Correlations** - связи между метриками

### Dimension Analysis
- Какие измерения важнее
- Как распределена информация
- Есть ли неиспользуемые измерения
- Баланс между измерениями

## Интерпретация

### ✅ Хорошие признаки

- Равномерное распределение точек
- Четкие кластеры похожих токенов
- Нормы в диапазоне 0.1-10
- Variance распределена по измерениям
- Нет огромных выбросов

### ❌ Проблемы

- Все точки в одном месте → модель не обучена
- Огромные выбросы → нестабильное обучение
- Вся variance в 1-2 измерениях → недоиспользование
- Очень большие/маленькие нормы → проблемы с масштабом

## Технические детали

### Методы снижения размерности

**UMAP** (рекомендуется)
- Сохраняет глобальную и локальную структуру
- Быстрее чем t-SNE
- Лучше для общего анализа

**PCA** (самый быстрый)
- Линейное преобразование
- Показывает основные направления variance
- Хорош для быстрой проверки

**t-SNE** (для кластеров)
- Отлично показывает группы
- Медленный на больших данных
- Не сохраняет глобальные расстояния

## Использование в браузере

1. Откройте любой `.html` файл в браузере
2. Взаимодействие:
   - **Левая кнопка мыши** + движение = вращение
   - **Колесико мыши** = зум
   - **Правая кнопка мыши** + движение = перемещение
   - **Наведение курсора** = информация о точке
3. Кнопки в правом верхнем углу:
   - 📷 Сохранить как изображение
   - 🔄 Сбросить вид
   - 🔍 Зум
   - ↔️ Перемещение

## Troubleshooting

### Модель не найдена
```bash
# Проверьте путь
ls models/tokenier/embedding_model.pth

# Укажите полный путь
python visualize_embedding_model.py --model "C:/path/to/model.pth"
```

### Слишком долго
```bash
# Используйте PCA и меньше точек
python visualize_embedding_model.py --method pca --sample-size 1000
```

### Out of memory
```bash
# Уменьшите sample-size
python visualize_embedding_model.py --sample-size 2000

# Используйте 2D
python visualize_embedding_model.py --2d --sample-size 3000
```

### Ошибка: Could not find embeddings
Модель должна содержать один из:
- `embedding.weight`
- `model_state_dict['embedding.weight']`
- Любой 2D тензор
