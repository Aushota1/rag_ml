## 📊 Что смотреть

### Начните с 3d_umap.html
Это главная визуализация:
- Интерактивный 3D график
- Каждая точка = токен
- Цвет = норма вектора
- Можно вращать, зумить, наводить курсор

**Управление:**
- 🖱️ Левая кнопка + движение = вращение
- 🖱️ Колесико = зум
- 🖱️ Правая кнопка + движение = перемещение
- 👆 Наведение = информация о токене

### Затем distribution_analysis.html
Статистика:
- Распределение норм векторов
- Средние значения
- Стандартные отклонения
- Корреляции

### И dimension_analysis.html
Технический анализ:
- Важность каждого измерения
- Распределение variance
- Min/Max значения

## 🔍 Что проверить

### ✅ Хорошие признаки
- [ ] Точки равномерно распределены
- [ ] Видны кластеры похожих токенов
- [ ] Нормы в диапазоне 0.1-10
- [ ] Variance распределена по измерениям
- [ ] Нет огромных выбросов

### ❌ Проблемы
- [ ] Все точки в одном месте → не обучена
- [ ] Огромные выбросы → нестабильное обучение
- [ ] Вся variance в 1-2 измерениях → недоиспользование
- [ ] Нормы >100 или <0.01 → проблемы с масштабом
