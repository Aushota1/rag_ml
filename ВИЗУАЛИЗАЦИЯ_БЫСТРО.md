# 🚀 Визуализация Embeddings - Быстрая шпаргалка

## Одна команда для всего

```bash
python visualize_embedding_model.py --full-report
```

Откройте `visualizations/*.html` в браузере.

## Основные команды

```bash
# Полный отчет (5 визуализаций)
python visualize_embedding_model.py --full-report

# Только 3D UMAP
python visualize_embedding_model.py

# Только 3D PCA (быстрее)
python visualize_embedding_model.py --method pca

# 2D визуализация
python visualize_embedding_model.py --2d

# Больше точек (детальнее)
python visualize_embedding_model.py --sample-size 10000

# Кастомная модель
python visualize_embedding_model.py --model path/to/model.pth
```

## Что получите

После `--full-report`:

1. **3d_umap.html** - 3D проекция (UMAP)
2. **3d_pca.html** - 3D проекция (PCA)
3. **2d_umap.html** - 2D проекция
4. **distribution_analysis.html** - Статистика
5. **dimension_analysis.html** - Анализ измерений

## Параметры

| Параметр | Значения | По умолчанию |
|----------|----------|--------------|
| `--method` | umap, pca, tsne | umap |
| `--sample-size` | 1000-20000 | 5000 |
| `--output-dir` | путь | visualizations |
| `--2d` | флаг | 3D |
| `--full-report` | флаг | одна визуализация |

## Быстрый тест

```bash
# Минимальный (30 сек)
python visualize_embedding_model.py --method pca --sample-size 1000

# Оптимальный (2-3 мин)
python visualize_embedding_model.py --full-report

# Максимальный (5-10 мин)
python visualize_embedding_model.py --full-report --sample-size 20000
```

## Интерактивность

В браузере:
- 🖱️ Левая кнопка = вращение
- 🖱️ Колесико = зум
- 🖱️ Правая кнопка = перемещение
- 👆 Наведение = информация

## Что смотреть

✅ **Хорошо:**
- Равномерное распределение
- Четкие кластеры
- Нормы 0.1-10

❌ **Плохо:**
- Все точки в куче
- Огромные выбросы
- Нормы >100 или <0.01

## Troubleshooting

```bash
# Модель не найдена
ls models/tokenier/embedding_model.pth

# Слишком долго
python visualize_embedding_model.py --method pca --sample-size 1000

# Out of memory
python visualize_embedding_model.py --2d --sample-size 2000
```
