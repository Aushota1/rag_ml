## 🎓 Обучение токенизатора и эмбеддингов

### Шаг 1: Обучение BPE токенизатора

BPE (Byte Pair Encoding) токенизатор разбивает текст на подслова.

```bash
# Базовое обучение
python train_bpe_tokenizer.py

# С параметрами
python train_bpe_tokenizer.py \
  --documents-path C:/Users/Aushota/Downloads/dataset_documents \
  --vocab-size 30000 \
  --output models/tokenier/tokenizer.pkl \
  --test
```

**Параметры:**
- `--documents-path`: Путь к PDF документам
- `--vocab-size`: Размер словаря (рекомендуется 20000-50000)
- `--output`: Путь для сохранения модели
- `--checkpoint-interval`: Интервал сохранения (по умолчанию 100)
- `--test`: Протестировать после обучения

**Время:** 30-60 минут (зависит от количества документов)

**Результат:** `models/tokenier/tokenizer.pkl`

### Шаг 2: Обучение Embedding Layer

Embedding layer создает векторные представления токенов.

```bash
# Базовое обучение
python train_embedding_layer.py

# С параметрами
python train_embedding_layer.py \
  --tokenizer-path models/tokenier/tokenizer.pkl \
  --data-dir C:/Users/Aushota/Downloads/dataset_documents \
  --output-path models/tokenier/embedding.pth \
  --embedding-dim 256 \
  --epochs 10 \
  --batch-size 32 \
  --device cpu
```

**Параметры:**
- `--tokenizer-path`: Путь к обученному токенизатору
- `--data-dir`: Директория с PDF документами
- `--output-path`: Путь для сохранения эмбеддингов
- `--embedding-dim`: Размерность эмбеддингов (128, 256, 512)
- `--max-seq-len`: Максимальная длина последовательности
- `--batch-size`: Размер батча
- `--epochs`: Количество эпох
- `--learning-rate`: Скорость обучения
- `--device`: Устройство (cpu или cuda)

**Время:** 30-60 минут (зависит от количества документов и epochs)

**Результат:** `models/tokenier/embedding.pth`

**Примечание:** Для обучения на GPU используйте `--device cuda`

## 🤖 Обучение классификаторов

После обучения токенизатора и эмбеддингов, обучите классификаторы:

### Все классификаторы сразу

```bash
python train_tokenier_models.py --model all
```

```bash

python train_tokenier_models.py --model relevance
```

**Время:** 10-20 минут для всех моделей

**Результаты:**
- `models/tokenier/document_classifier.joblib`
- `models/tokenier/question_classifier.joblib`
- `models/tokenier/relevance_classifier.joblib`

## 📊 Построение индекса

### Standard Mode (без tokenier)

Быстрое построение индекса без классификации:

```bash
python build_index.py --mode standard
```

**Особенности:**
- ✅ Быстро (2-5 минут)
- ✅ Не требует обучения моделей
- ✅ Структурная сегментация
- ⚠️ Базовое качество

### Tokenier Mode (с классификацией)

Построение индекса с классификацией и семантической сегментацией:

```bash
python build_index.py --mode tokenier
```

**Особенности:**
- ✅ Классификация типов документов
- ✅ Семантическая сегментация
- ✅ Улучшенное качество (+5-10%)
- ⚠️ Медленнее (5-15 минут)
- ⚠️ Требует обученных моделей

### Сравнение режимов

```bash
python compare_indexing_modes.py
```

Результаты сохраняются в `comparison_results.json`

## 🐛 Troubleshooting

### Проблема: "Tokenizer not found"

**Причина:** Файл `tokenizer.pkl` отсутствует

**Решение 1:** Скопировать из tokenier
```bash
cp C:\Users\Aushota\Desktop\tokenier\chekpoint.pkl models\tokenier\tokenizer.pkl
```

**Решение 2:** Обучить самостоятельно
```bash
python train_bpe_tokenizer.py
```

### Проблема: "Embeddings not found"

**Причина:** Файл `embedding.pth` отсутствует

**Решение 1:** Скопировать из tokenier
```bash
cp C:\Users\Aushota\Desktop\tokenier\chekpoint\best_model_epoch_20.pth models\tokenier\embedding.pth
```

**Решение 2:** Обучить самостоятельно
```bash
python train_embedding_layer.py
```

### Проблема: "No models trained"

**Причина:** Классификаторы не обучены

**Решение:**
```bash
python train_tokenier_models.py --model all
```

### Проблема: "Out of memory" при обучении

**Причина:** Недостаточно RAM

**Решение:**
```bash
# Уменьшите batch size
python train_embedding_layer.py --batch-size 16

# Или уменьшите embedding dim
python train_embedding_layer.py --embedding-dim 128
```

### Проблема: Медленное обучение

**Причина:** Обучение на CPU

**Решение:** Используйте GPU если доступен
```bash
python train_embedding_layer.py --device cuda
```

### Проблема: "No PDF files found"

**Причина:** Неправильный путь к документам

**Решение:**
```bash
# Проверьте путь
ls C:/Users/Aushota/Downloads/dataset_documents/*.pdf

# Укажите правильный путь
python train_bpe_tokenizer.py --documents-path /правильный/путь
```
