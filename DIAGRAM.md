# Диаграммы архитектуры RAG системы

## Общая архитектура

```
┌─────────────────────────────────────────────────────────────────┐
│                         RAG ML SYSTEM                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                   │
│  ┌──────────────────┐         ┌──────────────────┐             │
│  │  ЭТАП 1:         │         │  ЭТАП 2:         │             │
```

## Этап 2: Инференс

```
┌─────────────────────────────────────────────────────────────────┐
│                     INFERENCE PIPELINE                           │
└─────────────────────────────────────────────────────────────────┘

    💬 User Question
           │
           ▼
    ┌──────────────┐
    │    QUERY     │  ← Synonym expansion
    │   REWRITER   │     Simplification
    └──────┬───────┘     Question→Statement
           │
           │ Original + 2-3 variants
           ▼
    ┌──────────────────────────────────┐
    │      HYBRID RETRIEVER            │
    ├──────────────────────────────────┤
    │                                  │
    │  ┌─────────────┐  ┌───────────┐ │
    │  │  VECTOR     │  │   BM25    │ │
    │  │  SEARCH     │  │  SEARCH   │ │
    │  │             │  │           │ │
    │  │ Top-20      │  │ Top-20    │ │
    │  └──────┬──────┘  └─────┬─────┘ │
    │         │                │       │
    │         └────────┬───────┘       │
    │                  │               │
    │         Combine (α=0.5)          │
    │                  │               │
    │         Deduplicate              │
    └──────────────────┬───────────────┘
                       │
                       │ Top-40 candidates
                       ▼
    ┌──────────────────────────────────┐
    │         RERANKER                 │
    │  (Cross-Encoder)                 │
    │                                  │
    │  Pairwise scoring:               │
    │  (query, doc₁), (query, doc₂)... │
    └──────────────────┬───────────────┘
                       │
                       │ Top-5 reranked
                       ▼
    ┌──────────────────────────────────┐
    │    THRESHOLD VALIDATOR           │
    │                                  │
    │  max_score >= 0.3 ?              │
    │    ├─ Yes → has_info = true      │
    │    └─ No  → has_info = false     │
    └──────────────────┬───────────────┘
                       │
           ┌───────────┴───────────┐
           │                       │
      has_info=true          has_info=false
           │                       │
           ▼                       ▼
    ┌──────────────┐      ┌──────────────┐
    │   ANSWER     │      │    EMPTY     │
    │  GENERATOR   │      │   ANSWER     │
    │              │      │              │
    │ Context +    │      │ "No info     │
    │ LLM/Rules    │      │  found"      │
    └──────┬───────┘      └──────┬───────┘
           │                     │
           └──────────┬──────────┘
                      │
                      ▼
    ┌──────────────────────────────────┐
    │      TELEMETRY COLLECTOR         │
    │                                  │
    │  • TTFT (time to first token)    │
    │  • Total time                    │
    │  • Token usage                   │
    │  • Retrieved pages               │
    └──────────────────┬───────────────┘
                       │
                       ▼
    📤 JSON Response
    {
      "answer": {...},
      "telemetry": {...}
    }
```
