# 🎨 Embedding Model Visualizations

## Созданные визуализации

Все файлы интерактивные - откройте в браузере!

### 1. 3d_umap.html
**3D UMAP проекция эмбеддингов**
- Лучше всего показывает структуру данных
- Сохраняет локальные и глобальные отношения
- Цвет = норма вектора

### 2. 3d_pca.html
**3D PCA проекция**
- Линейное снижение размерности
- Показывает основные направления variance
- Explained variance: ~33%

### 3. 2d_umap.html
**2D UMAP проекция**
- Плоская визуализация
- Легче анализировать кластеры
- Больше точек видно одновременно

### 4. distribution_analysis.html
**Статистический анализ**
- Распределение норм векторов
- Распределение средних значений
- Распределение стандартных отклонений
- Корреляция между метриками

### 5. dimension_analysis.html
**Анализ по измерениям**
- Среднее значение каждого измерения
- Стандартное отклонение по измерениям
- Min/Max значения
- Variance по измерениям

## Информация о модели

- **Vocabulary size:** 508 токенов
- **Embedding dimension:** 256
- **Total parameters:** 130,048

## Как использовать

1. Откройте любой `.html` файл в браузере (Chrome, Firefox, Edge)
2. Взаимодействие с 3D графиками:
   - **Левая кнопка мыши** + движение = вращение
   - **Колесико мыши** = зум
   - **Правая кнопка мыши** + движение = перемещение
   - **Наведение курсора** = информация о точке

## Интерпретация

### ✅ Хорошие признаки
- Равномерное распределение точек
- Четкие кластеры похожих токенов
- Нормы в разумном диапазоне
- Variance распределена по измерениям

### ❌ Проблемы
- Все точки в одном месте
- Огромные выбросы
- Вся variance в 1-2 измерениях
- Очень большие/маленькие нормы

## Пересоздание визуализаций

```bash
# Полный отчет
python visualize_embedding_model.py --full-report

# Только 3D UMAP
python visualize_embedding_model.py

# Только 2D
python visualize_embedding_model.py --2d

# С другим методом
python visualize_embedding_model.py --method pca
```
