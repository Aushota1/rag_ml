## Поток данных

### 1. Классификация документов

```
PDF → Parser → Текст (первые 1000 символов)
                │
                ▼
         ┌──────────────┐
         │ BPE Tokenizer│
         └──────┬───────┘
                │ token_ids
                ▼
         ┌──────────────┐
         │  Embedding   │
         │    Layer     │
         └──────┬───────┘
                │ embeddings (batch, seq, 256)
                ▼
         ┌──────────────┐
         │ Mean Pooling │
         └──────┬───────┘
                │ vector (256)
                ▼
         ┌──────────────┐
         │  Classifier  │
         │ (RF/XGBoost) │
         └──────┬───────┘
                │
                ▼
         [law/case/regulation]
                │
                ▼
         Добавить в metadata
```

### 2. Классификация вопросов

```
Вопрос → BPE Tokenizer → Embedding → Mean Pool → Classifier
                                                      │
                                                      ▼
                                            [factual/procedural/...]
                                                      │
                                                      ▼
                                            Параметры поиска:
                                            - alpha: 0.3-0.7
                                            - top_k: 15-50
                                            - threshold: 0.25-0.5
                                                      │
                                                      ▼
                                            Применить к Retriever
```

### 3. Семантическая сегментация

```
Текст документа
    │
    ▼
Создать скользящие окна (200 слов, overlap 50)
    │
    ▼
Для каждого окна:
    │
    ├─→ BPE Tokenizer → Embedding → Mean Pool → vector_i
    │
    ▼
Вычислить косинусное расстояние между соседними окнами:
    │
    ├─→ distance_i = 1 - cosine_similarity(vector_i, vector_i+1)
    │
    ▼
Найти пики (границы):
    │
    ├─→ threshold = mean(distances) + std(distances)
    ├─→ boundaries = [i where distance_i > threshold]
    │
    ▼
Разбить текст по границам → Чанки
```

### 4. Гибридные эмбеддинги

```
Текст
    │
    ├─────────────────────┬─────────────────────┐
    │                     │                     │
    ▼                     ▼                     ▼
sentence-transformers  BPE Tokenizer      (параллельно)
    │                     │
    ▼                     ▼
  (384d)              Embedding
                          │
                          ▼
                      Mean Pool
                          │
                          ▼
                        (256d)
    │                     │
    └──────────┬──────────┘
               │
               ▼
        Concatenate
               │
               ▼
            (640d)
               │
               ▼
        Linear Projection
               │
               ▼
            (384d)
               │
               ▼
        Нормализация L2
               │
               ▼
        В FAISS индекс
```

### 5. Классификация релевантности

```
Вопрос + Чанк
    │
    ▼
Конкатенация: "[Q] {question} [SEP] [C] {chunk}"
    │
    ▼
BPE Tokenizer (первые 1000 символов)
    │
    ▼
Embedding Layer
    │
    ▼
Mean Pooling
    │
    ▼
Classifier (Binary: relevant/not_relevant)
    │
    ▼
Probability of relevance
    │
    ▼
Filter if prob < threshold (0.5)
```

### Зависимости между компонентами

```
┌─────────────────────────────────────────────────────────────┐
│                    ЗАВИСИМОСТИ                              │
└─────────────────────────────────────────────────────────────┘

BPE Tokenizer (tokenier/BPE_STUCTUR.py)
    │
    ├─→ Embedding Layer (tokenier/EMBEDDING_LAYER/)
    │       │
    │       ├─→ Document Classifier
    │       ├─→ Question Classifier
    │       ├─→ Semantic Chunker
    │       ├─→ Hybrid Embedding
    │       └─→ Relevance Classifier
    │
    └─→ Все компоненты интеграции

sentence-transformers
    │
    └─→ Hybrid Embedding (комбинация с tokenier)

scikit-learn / XGBoost
    │
    ├─→ Document Classifier
    ├─→ Question Classifier
    └─→ Relevance Classifier

FAISS
    │
    └─→ HybridIndexer (с/без Hybrid Embedding)
```
