# 📋 Полезные команды

## 🚀 Быстрый старт

### Standard Mode (5 минут)
```bash
pip install -r requirements.txt
cp .env.example .env
# Отредактируйте .env
python build_index.py --mode standard
python api.py
```

### Tokenier Mode (30 минут)
```bash
python check_tokenier_setup.py
python train_tokenier_models.py --model all
python build_index.py --mode tokenier
python api.py
```

## 🗂️ Копирование файлов tokenier

### Windows PowerShell
```powershell
# Создать директорию
New-Item -ItemType Directory -Force -Path models\tokenier

# Скопировать токенизатор
Copy-Item "C:\Users\Aushota\Desktop\tokenier\chekpoint.pkl" "models\tokenier\tokenizer.pkl" -Force

# Скопировать эмбеддинги
Copy-Item "C:\Users\Aushota\Desktop\tokenier\chekpoint\best_model_epoch_20.pth" "models\tokenier\embedding.pth" -Force

# Проверить
ls models\tokenier\
```

### Linux/Mac
```bash
# Создать директорию
mkdir -p models/tokenier

# Скопировать токенизатор
cp ~/tokenier/chekpoint.pkl models/tokenier/tokenizer.pkl

# Скопировать эмбеддинги
cp ~/tokenier/chekpoint/best_model_epoch_20.pth models/tokenier/embedding.pth

# Проверить
ls -lh models/tokenier/
```

## 🎯 Полезные алиасы

Добавьте в `~/.bashrc` или `~/.zshrc`:

```bash
# RAG алиасы
alias rag-build-std="python build_index.py --mode standard"
alias rag-build-tok="python build_index.py --mode tokenier"
alias rag-check="python check_tokenier_setup.py"
alias rag-train="python train_tokenier_models.py --model all"
alias rag-compare="python compare_indexing_modes.py"
alias rag-api="python api.py"
alias rag-test="python quick_test.py"
```

Использование:
```bash
rag-check      # Проверить настройку
rag-train      # Обучить модели
rag-build-tok  # Построить индекс с tokenier
rag-api        # Запустить API
```
