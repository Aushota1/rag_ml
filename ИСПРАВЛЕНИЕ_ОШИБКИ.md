# Исправление ошибки OpenAI API

## Проблема

Вы видите:
```
Warning: Failed to initialize OpenAI client: The api_key client option must be set
```

## Причина

Система настроена на использование LLM (gpt-4o-mini), но не может найти API ключ OpenAI.

## Быстрое решение (выберите один вариант)

### ✅ Вариант 1: Добавить API ключ (рекомендуется)

1. Откройте файл: `C:\Users\Aushota\Desktop\rag_ml\.env`

2. Найдите строку:
   ```env
   OPENAI_API_KEY=sk-your-actual-api-key-here
   ```

3. Замените на ваш реальный ключ:
   ```env
   OPENAI_API_KEY=sk-proj-abc123xyz...
   ```

4. Сохраните и запустите снова

**Где взять ключ?** → https://platform.openai.com/api-keys

### ✅ Вариант 2: Отключить LLM (бесплатно)

1. Откройте файл: `C:\Users\Aushota\Desktop\rag_ml\.env`

2. Найдите строку:
   ```env
   USE_LLM=true
   ```

3. Измените на:
   ```env
   USE_LLM=false
   ```

4. Сохраните и запустите снова

## Что происходит сейчас?

Система работает в режиме **fallback** - использует эвристики вместо LLM.
Это работает, но качество ответов будет ниже.
