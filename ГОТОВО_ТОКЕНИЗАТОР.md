```bash

# 2. Обучить токенизатор
python train_bpe_tokenizer.py --corpus data/corpus.txt --vocab-size 30000 --test
```

## Параметры обучения

### Размер словаря (`--vocab-size`)
- **10000** - быстро, но низкое качество
- **30000** - оптимально (по умолчанию)
- **50000** - медленно, но высокое качество

### Чекпоинты (`--checkpoint-interval`)
- Сохраняет прогресс каждые N итераций
- По умолчанию: 100
- Можно прервать обучение (Ctrl+C) и продолжить позже
